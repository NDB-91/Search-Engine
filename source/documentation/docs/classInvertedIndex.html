Detailed Description

Implements the IIndex interface using an inverted index structure.

The InvertedIndex class provides methods to index documents, remove document indices, and query the index for relevant documents based on a search query. It uses an unordered map to store the mapping between terms and the set of document identifiers containing those terms.
