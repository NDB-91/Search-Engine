Detailed Description

Implements the ITermFrequency and IInverseDocumentFrequency interfaces to calculate TF-IDF values.

The TFIDF class provides methods to calculate Term Frequency (TF), Inverse Document Frequency (IDF), and the combined TF-IDF value for terms in documents.
